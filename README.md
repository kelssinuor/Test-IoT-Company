# IoT Light Sensor System with Arduino

## Description
This project simulates a light sensor system using Arduino. The system reads light intensity (LUX) and controls LEDs, LCD display, and buzzer based on specific conditions.

## Features
- Red LED ON when LUX < 500
- Blue LED ON when LUX > 500
- LCD displays real-time LUX value
- Buzzer activates when button is pressed AND LUX > 500

## Components
- Arduino UNO
- Light Sensor (LDR)
- LCD I2C
- LED (Red & Blue)
- Buzzer
- Push Button
